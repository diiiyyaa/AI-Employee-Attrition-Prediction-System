# 🤖 AI Employee Attrition Prediction & Analytics Dashboard

An end-to-end Machine Learning application that predicts employee attrition using the IBM HR Analytics dataset. The project includes an interactive Streamlit dashboard, predictive analytics, and HR insights to support data-driven employee retention strategies.

---

## 🚀 Live Demo

🔗 https://ai-employee-attrition-prediction-system-ggxxpxxhr8cukb76wthqtu.streamlit.app/

---

## 📌 Features

- 📊 Interactive HR Analytics Dashboard
- 🤖 Employee Attrition Prediction
- 📈 Business Insights & Workforce Analytics
- 📉 Interactive Plotly Visualizations
- 🧠 Machine Learning Pipeline
- 📄 Download Prediction Report
- 🌙 Modern Dark-Themed User Interface
